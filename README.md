# RoKiGL-Qtテスト
RoKiGLをQtで動かすテスト

# ビルド方法
```
$rokigl-pro-gen
$mkdir build
$cd build
$qmake ..
$make
```

# 要求
* RoKi Ver.1.0.0a (http://www.mi.ams.eng.osaka-u.ac.jp/software/roki.html)
* RoKi-GL Ver.1.0.0-beta5 (http://www.mi.ams.eng.osaka-u.ac.jp/software/roki.html)
* Qt Ver 5.6 (https://www.qt.io/download-open-source/)
